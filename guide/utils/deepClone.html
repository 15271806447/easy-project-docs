# deepClone 深度克隆

深度克隆：变量的属性均完全复制，与原对象完全脱离，所有对于新对象的修改都不会反映到原对象中。

# 按需引入

//引入方法
import deepClone from 'easy-project-vue/dist/utils/deepClone.js'
//使用方法
let obj2 = deepClone(obj)

# 何时使用

值类型变量与引用类型变量

如果我们有两个值类型的变量(如字符串、数字、布尔值），将其中一个变量的值赋给另一个，实际上会创建该值的一个副本，这个副本与原来的值没有什么关系。

——这意味着改变其中一个的值不会影响另一个变量的值。

如果是两个引用类型的变量（如对象、数组），其中一个变量的值赋给另一个的话，并没有创建值的副本，而是使两个变量指向同一个内存地址。

——这意味着改变对象的值会同时影响两个变量。

要真正地创建引用类型的副本，我们必须 深度克隆(deepClone) 变量指向的对象。

# 添加版本

0.3.0

# 参数

参数 说明 类型 默认值
target 要被拷贝的对象 * -

# 返回值

(*): 返回 深度克隆 后的对象。

# 用法示例

import deepClone from 'easy-project-vue/dist/utils/deepClone.js'

 let s1 = Symbol('s1')
 let obj = {
     a: '100',
     b: undefined,
     c: null,
     d: Symbol(2),
     e: /^\d+$/,
     f: new Date(),
     g: true,
     arr: [10, 20, 30],
     school: {
         name: 'cherry',
         [s1]: 's1'
     },
     fn: function fn() {
         console.log('fn')
     }
 }
 obj.h = obj
 let obj2 = deepClone(obj)
 console.log('克隆', obj, obj2)
 obj2.a = 600
 console.log('修改obj2不会影响obj的值', obj, obj2)
