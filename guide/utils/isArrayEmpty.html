# isArrayEmpty 数组判空

判断数组为空：传递参数为数组形式的数据才会正确执行，其他形式的数据都会返回false。

# 按需引入

//引入方法
import isArrayEmpty from 'easy-project-vue/dist/utils/isArrayEmpty.js'
//使用方法
let arr = isArrayEmpty(array)

# 何时使用

对数组进行迭代时需判断传入的数据是否为空

# 添加版本

0.3.0

# 参数

参数 说明 类型 默认值
array 要判断的数组 * -

# 返回值

(*): 返回 该数据是否为空数组的布尔值。

# 用法示例

import isArrayEmpty from 'easy-project-vue/dist/utils/isArrayEmpty.js'

let arr = [
	{name: '张三', age: '21'},
	{name: '李四', age: '23'},
	{name: '王五', age: '25'}
]
if (isArrayEmpty(arr)) {
	console.log("传递的数组是个空数组！")
}
console.log(arr)
return arr
