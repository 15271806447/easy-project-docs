# highlightKeywords 高亮文字

高亮文字：对一个字符串中的文字进行高亮设置，高亮文字可以设置一个或多个。

# 按需引入

//引入方法
import highlightKeywords from 'easy-project-vue/dist/utils/highlightKeywords.js'
//使用方法
let highText = highlightKeywords(val, keywords, color)

# 何时使用

当需要对一段文字内的关键字进行高亮显示时设置使用，使用时需注意:处理后的高亮文本是带有HTML标签的文本，需要使用v-html标签解析

# 添加版本

0.3.0

# 参数

参数 说明 类型 默认值
val 传入的原始文本 String -
keywords 单个关键词字符串或字符串数组 String/Array -
color 设置高亮色 String '#FF4441'

# 返回值

(*): 返回 带有HTML标签的字符串，使用时需要使用v-html指令解析文本。

# 用法示例

高亮文字显示

webpack详细的配置选项 (只对一个关键字进行高亮显示)
webpack详细的配置选项 (对多个关键字进行高亮显示) 
方法一：使用变量进行处理
<template>
  <div>
    <div class="content">
      <span v-html="highStringText">
      </span>
      (只对一个关键字进行高亮显示)<br>
      <span v-html="highArrayText">
      </span>
      (对多个关键字进行高亮显示)
    </div>
  </div>
</template>
<script>
import highlightKeywords from 'easy-project-vue/dist/utils/highlightKeywords.js'

export default {
  data() {
    return {
      highStringText: 'webpack详细的配置选项',
      highArrayText: 'webpack详细的配置选项'
    }
  },
  mounted() {
    this.highString()
    this.highArray()
  },
  methods: {
    highString() {
      let highString = this.highStringText
      this.highStringText = highlightKeywords(highString, 'webpack', '#008AFF')
    },
    highArray() {
      let highString = this.highArrayText
      this.highArrayText = highlightKeywords(highString, ['webpack', '配置'], '#008AFF')
    }
  }
}

</script>

方法二：'哪里用到变哪里',直接在标签内部使用方法！
<template>
  <p v-html="highlightKeywords('webpack详细的配置选项', ['webpack','配置'], '#008AFF')"></p>
  <p v-html="highlightKeywords('webpack详细的配置选项', 'webpack', '#008AFF')"></p>
</template>
