# space 间距

设置组件之间的间距（flex布局）。

# 何时使用

避免组件紧贴在一起，拉开统一的空间。

  • 适合行内元素的水平间距。
  • 可以设置各种水平对齐方式。

# 配置参数

属性 说明 类型 可选参数 默认值
justify 主轴方向上的对齐方式 string start/ end / center / space-around / space-between start
align 侧轴方向上的对齐方式 string start/ end / center / stretch / baseline -
direction 主轴方向 string row / column row
size 每个子元素之间的间距 string css中所允许的单位都可使用："20px"、"20rem"、"2%"等 -
