# stretchheight 余屏高度

使得一个dom元素的高度永远等于屏幕剩下的高度。

# 按需引入

import Stretchheight from 'easy-project-vue/dist/directives/stretchheight.js'

Vue.use(Stretchheight);

# 何时使用

下面是一个常见的界面布局：

首页
菜单1
菜单2
菜单3
我是主内容区块，并希望高度等于屏幕剩下的高度。

一个常见的需求是，我们希望给主内容区块设为占满屏幕剩下的空间，但这使用px是无法做到的，因为你无法确定用户电脑的尺寸。

常见的做法是用100vh减去此dom上面所有区块高度之和，即height: calc(100vh - 50px);

但那样计算太麻烦，需要一个个地核对上面所有的div高度，使用v-stretchheight指令可以简洁的解决这个问题。

# 用法示例

v-stretchheight的参数是一个回调函数，函数参数包括计算好的高度，及指令绑定的$el实例，常见用法如下：


<div v-stretchheight="({ height, el }) => { el.style.height = height + 'px' }">
我的高度永远等于屏幕剩下的高度
</div>

如果你只希望限定最小高度，那么可以改为设置minHeight属性


<div v-stretchheight="({ height, el }) => { el.style.minHeight = height + 'px' }">
我的高度永远等于屏幕剩下的高度
</div>

还有一种用法是拿到所需高度，并赋值给组件的props。 如下面用tableHeight存储计算高度，并赋值给antd表格组件的scroll属性。


<template>
  <a-table
      :scroll="{x: 'calc(60px + 100%)',y:tableHeight - 100 + 'px'}"
      v-stretchheight="({ height }) => { this.tableHeight = height }"
      :style="{height: tableHeight - 10 + 'px'}"
  >
  </a-table>
</template>
<script>
export default {
  data() {
    return {
      tableHeight: 400
    }
  }
}
</script>
