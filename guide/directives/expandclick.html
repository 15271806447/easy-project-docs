# expandclick 扩展热区

扩展一个dom元素的热区，即点击、悬浮等范围扩大。

# 按需引入

import Expandclick from 'easy-project-vue/dist/directives/expandclick.js'
Vue.use(Expandclick);

# 何时使用

让我们来看这样一个场景，现在我们给下面这一按钮绑定点击事件，想给一个简单的小按钮扩大其热区在四个方向上均向外扩张20px，点击下面的文字试试看：

<template>
    <div>
        <h3>扩大点击热区域前</h3>
        <div class="content" @click="click">
            请点击我
        </div>
        <h3>扩大点击热区域后，可点击红线内的区域触发事件</h3>
        <div style="width: 80px;padding: 20px;border: 1px solid red">
            <div class="content" @click="click"
                 v-expandclick:20>
                请点击我
            </div>
        </div>
    </div>
</template>
<script>

    export default {
        methods: {
            click() {
                alert('我被点击了')
            }
        }
    }
</script>
<style scoped>
    .content {
        background: pink;
        width: 80px;
        height: 40px;
        line-height: 40px;
        text-align: center;
    }
</style>

# 配置项

参数 说明 类型 默认值
expandSize 默认的扩展热区半径(单位px) string 10

注册配置

import Expandclick from 'easy-project-vue/dist/directives/expandclick.js'

let Options = { expandSize:20 }
Vue.use(Expandclick,Options);

# 用法示例

<template>
    <div>
           <h3>v-expandclick:20 一个值代表【上下左右】扩大</h3>
           <div style="width: 80px;padding: 20px;border: 1px solid red">
               <div class="content" @click="click"
                    v-expandclick:20>
                   请点击我
               </div>
           </div>
           <h3>v-expandclick:20*30 两个值对应【上下】【左右】</h3>
           <div style="width: 120px;padding: 20px 30px;border: 1px solid red">
               <div class="content" style="width: 120px;" @click="click"
                    v-expandclick:20*30>
                   请点击我
               </div>
           </div>
           <h3>v-expandclick:20*30*40 两个值对应三个值对应 【上】【左右】【下】</h3>
           <div style="width: 120px;padding: 20px 30px 40px;border: 1px solid red">
               <div class="content" style="width: 120px;" @click="click"
                    v-expandclick:20*30*40>
                   请点击我
               </div>
           </div>
           <h3>v-expandclick:10*20*30*40 四个值对应 【上】【右】【下】【左】</h3>
           <div style="width: 120px;padding:10px 20px 30px 40px;border: 1px solid red">
               <div class="content" style="width: 120px;" @click="click"
                    v-expandclick:10*20*30*40>
                   请点击我
               </div>
           </div>
           <h3>总结一下，规则就跟css的margin、padding的值填写规则一样。</h3>    
    </div>
</template>
<script>
  export default {
       methods: {
          click() {
            alert('我被点击了')
          }
        }
  }
</script>
