# maskclosable 蒙层点击

监听点击一个div以外的区域所触发的点击事件

# 按需引入

import Maskclosable from 'easy-project-vue/dist/directives/maskclosable.js'

Vue.use(Maskclosable);

# 何时使用

  • 例如弹出一个弹窗，我们称弹窗以外的区域为蒙层，常见的需求是点击蒙层后，关闭弹窗。

  • 当下拉框出现时，点击下拉区域以外的区域后，应该收起下拉框。

针对这些场景，我们可以使用v-maskclosable指令来监听蒙层点击事件。

# 用法示例

下面来看一个简单的下拉框示例，点击选择框可出现下拉框。

在使用v-maskclosable指令绑定关闭事件后，点击一个div以外的区域，即粉色区域，将触发我们的事件，关闭掉下拉框。

请你试试看： 


<template>
  <div style="padding: 50px;background: pink">
    <div style="width: 200px" v-maskclosable="close">
      <div class="content" @click="showOptions = true">请选择一项</div>
      <div class="select" v-show="showOptions">
        <div class="select-item">选项1</div>
        <div class="select-item">选项2</div>
        <div class="select-item">选项3</div>
        <div class="select-item">选项4</div>
      </div>
    </div>
  </div>
</template>
<script>

export default {
  data() {
    return {
      showOptions: true
    }
  },
  methods: {
    close() {
      this.showOptions = false
    }
  }
}
</script>
<style scoped>
.content {
  background: #fff;
  width: 100%;
  height: 40px;
  line-height: 40px;
  text-align: center;
}

.select {
  background: #fff;
  width: 100%;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15);
}

.select-item {
  width: 100%;
  height: 30px;
  line-height: 30px;
  text-align: center;
  border-bottom: 1px solid rgba(0, 0, 0, 0.45);
}

.select-item:first-child {
  border-top: 1px solid rgba(0, 0, 0, 0.45);
}
</style>
