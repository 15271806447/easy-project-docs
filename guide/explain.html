# 介绍

Easy Project Vue，是一款基于`Vue2`开发的`实用插件`集合，包括`组件`、`指令`、`函数`三大模块，致力于提供给程序员愉悦的开发体验。

# 项目特点

  • 🎨 提炼自实战项目中 高度复用 的代码集。
  • 🌍 开箱即用的高质量 Vue 组件、指令、函数。
  • 丰富的 代码片段 和 编码指引 。
